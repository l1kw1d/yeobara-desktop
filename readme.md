# yeobara-desktop

> yobara(`여봐라` in Korea), which mean is almost the same meaning of `Excuse me. I'm here` in English


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
